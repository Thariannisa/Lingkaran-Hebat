# Lingkaran Hebat

## Run

1. clone repo
```
    $ git clone <link repo>
```
2. install requirement modules
```
    $ pip install -r requirements.txt
```
3. migrate database
```
    $ python manage.py migrate
```
4. run server
```
    $ python manage.py runserver
```

## Requirement

- [mongoDb](https://www.mongodb.com/products/compass)
- [python](https://www.python.org/)
